Theme Installation

As a perquisite of using the theme, you need to have a working Shopify store. You can visit https://www.shopify.com to create a new store or run a trial.

Login to your Shopify online store admin panel.

Navigate to Left Menu -> Online Stores -> Themes:

Click on ‘Upload Theme’ button.

From newly opened ‘Upload theme’ dialog, click on ‘Browse’ button and select your Tishfy2 theme zip file.

Then click on ‘Upload file’ button

For new Tishfy2 theme in list of themes, click on ‘Actions’, and then click on ‘Publish’ button.

From the preview theme dialog click on ‘Publish’ button to confirm.

Demo Import

Products data can be imported:

  1. Login to your Shopify Admin
  2. Navigate to Left Menu - Products
  3. Click on Import
  4. From the opened dialog, select and upload file tishfy2-demo/Demo/tisfy2_demo_products.csv

Theme default settings (i.e. logo, slider content, brands, sections) are same as our theme demo at https://tishonator.com/shopify-demo/tishfy2/. Please check 'Theme Customizations' section how to customize these sections.

Theme Customizations

Header Logo

Navigate to Admin Panel -> Left Menu -> Online Store -> Themes

Tishfy2 -> Click on 'Customize' button

Click on 'Header' section

Select and upload image for 'Logo image'

You can also change the content for 'Header Top Left Content' and 'Header Top Right Content' which are displayed in website header top as HTML

Homepage Slider

Navigate to Admin Panel -> Left Menu -> Online Store -> Themes

Tishfy2 -> Click on 'Customize' button

Click on 'Homepage Slider' section

Update texts for Titles, content, and upload images for the slides

Main Menu

You can use drop-down menus to group products, collections, or pages together and make it easier for customers to navigate your online store. For example, if you have a lot of products then you might want to add them to collections and then use a drop-down menu from the main menu to organize the collections. This can help a customer to find the type of products that they're looking for.

Navigate to Admin Panel -> Left Menu -> Online Store -> Navigation

Click on 'Add menu' link to create a new menu

You can build drop-down menu by creating or moving menu items so that they are "nested" below a top-level item. The top-level item appears in the main menu on your online store, and the nested menu items appear in a drop-down menu. The top-level item can have up to two levels of nested drop-down menus:

The theme will display nested items as drop-down menus from the main menu, and some themes will display nested items as drop-down menus in other locations.

Once the menu is created, Navigate to Admin Panel -> Left Menu -> Online Store -> Themes

Tishfy2 -> Click on 'Customize' button

Click on 'Header' section

Set the menu from prevous step as 'Main menu'

Footer Columns

Navigate to Admin Panel -> Left Menu -> Online Store -> Themes

Tishfy2 -> Click on 'Customize' button

Click on 'Footer' section

Update the content (as HTML) for Footer columns and Social Site URLs

Save changes

Contact us Page

Navigate to Admin Panel -> Online Store -> Pages

Click on 'Add Page' button

Enter title i.e. 'Contacts' and (optionally) content

In right menu, select 'contact' for 'Theme template'

Save changes and publish the page.
